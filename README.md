# THE ESPER STACK

**System Architecture v1.0**

The Esper Stack is a trinary computing architecture designed to process semantic topology rather than binary logic.

## The Triad

1.  **Visual Layer (Input/Output):** [PICTOGRAM](modules/pictogram)
    * *Protocol:* PSH-256
    * *Role:* Immutable storage and transmission of semantic state.

2.  **Semantic Layer (Processing):** [VSE](modules/vse)
    * *Protocol:* Vector-Space Esperanto
    * *Role:* The "machine code" of meaning. High-dimensional vector mapping.

3.  **Execution Layer (Time):** [CHRONOCORE](modules/chronocore)
    * *Protocol:* Dissonance Engine
    * *Role:* The CPU. Manages narrative tension, pacing, and causality.
